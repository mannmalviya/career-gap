# Career-Gap

A web app that helps users close the gap between their current skills and a target job, then (later) hands off the boilerplate of applying to an AI agent.

## Style rules

- **Never use em dashes (—) anywhere.** Not in UI copy, code comments, commit messages, or docs. Use a hyphen with spaces (` - `), a colon, or restructure the sentence.
